
# Herb Share

This app serves as a space for users to share and view herbal recipes. Users can add recipes and comment on other recipes.

### Getting Started:
Once you've launched the app, you will be directed to login using Google Authorization.  
After you've logged in, you will be taken to the all recipes page. From there you can add recipes, view details, and comment on recipes.
 
Try it out --> [Herb Share](http://herb-share.herokuapp.com/)
